${Counter}

Click to increase or decrease the current value. As the value grows, the waiting time gets longer.

Button Description Waiting Time Feature
+1 add 1 to the current value |current value| × 100 msec
-1 subtract 1 from the current value |current value| × 100 msec
+10 add 10 to the current value |current value| × 10 msec pro
-10 subtract 10 from the current value |current value| × 10 msec pro

To use pro feature, you must sign in.
